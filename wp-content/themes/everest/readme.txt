Estrutura de páginas e arquivos do Wordpress


>>>> Páginas <<<<<

----> archive.php
    - página do blog (exibe a lista de posts).

----> search.php
    - página de exibição dos resultados de pesquisa.

----> single.php
    - página interna do post.

----> page.php
    - define oo template das páginas estáticas.


>>>>> Arquivos <<<<<

----> readme.txt
    - informações sobre os direitos de utilização do tema e infos relevantes ao usuário quanto a instalação do tema, etc.

----> screenshot.png
    - define a capa do tema.

----> favicon.php
    - define o icone na aba do navegador.
